## What's New

- **Cover shadows & rounded corners** — book covers and folder covers now have a consistent drop shadow and rounded corners across the homescreen and file browser.
- **Library router via BrowseMeta** — Library button can open All Books, By Series, By Author, or By Tags directly; mode is remembered across sessions.
- **GrimmLink shelf sync** — synced books appear immediately in BrowseMeta views without manual refresh.
- **Clock fix** — date text no longer gets cut off; font sizes and layout match reference values.
- **Performance** — removed unused requires across the plugin; startup is faster on slow eMMC devices.
